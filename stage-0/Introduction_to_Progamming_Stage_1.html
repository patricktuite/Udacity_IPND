The Basics of the Web and HTML

How does the web work?

Computers communicate with the internet via http. Requests are sent to servers that send an html file that is interpretted by a web browser for the user. A webpage is a text document written in HTML that is interpreted by a web browser.

What is HTML?

HTML (hypertext markup language) is the language that tells webpages what to display, how it looks, and where it links. The markup tells it what it looks like and is made up of tags, opening and closing tag make the element. HTML attributes are a property of an element. It's often a good idea to make your closing tag as soon as you make your opening tag so that you don't cause any unexpected problems.

What does it mean to say "Computers are Stupid"?

There is a need to write in the correct manner for the computer to understand and execute a programming language to have the desired effect. This is because computers follow a set of rules when executing a program. This is called syntax.

HTML Markup

HTML documents are made up of elements which are made of tags surrounding contents which can be text or other tags. HTML elements can have attributes. An example of an attribute is the "href" attribute contained within the anchor tag which defines a link. The attribute in this case is the link address. For example the element with anchor tag that contains the attribute href=www.udacity.com and content "Udacity" would result in the link Udacity.

Inline and Block Elements

There is an important distinction between these two elements. Block elements render an invisible "block" that can have a height and width. Both affect whitespace which is affected by elements as opposed to actual spaces in the code.
